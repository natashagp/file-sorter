# File Sorter

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/file-sorter.git
```

### Step 2

Change the directory to file-sorter

```shell
cd file-sorter
```

### Step 3

Run project

```shell
python main.py
```
